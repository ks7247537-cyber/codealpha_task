# 🤖 Smart Rule-Based Chatbot

A professional Python-based Rule-Based Chatbot developed as part of the CodeAlpha Python Programming Internship.

The chatbot interacts with users using predefined intents and responses while maintaining chat history and providing useful utilities such as date, time, calculator functions, jokes, and motivational quotes.

---

## 🚀 Features

- Greeting responses
- Farewell responses
- Help command
- Date information
- Time information
- Calculator mode
- Motivational quotes
- Random jokes
- User name memory
- Chat history storage
- Session statistics
- Input validation
- Error handling
- Clean CLI interface

---

## 🛠 Technologies Used

- Python 3
- Functions
- Dictionaries
- Loops
- Conditional Statements
- File Handling
- Date & Time Module
